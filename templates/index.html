Features

  • Scan complete subnets or specific IP ranges
  • Password or SSH key authentication
  • Execute custom commands or use templates
  • Collect detailed server information
  • Advanced filtering of results
  • Export results in various formats
  • Multi-threading for parallel processing

How to Use

Enter subnets in CIDR notation (e.g., 192.168.1.0/24), IP ranges (e.g., 192.168.1.1-192.168.1.10), or import from a CSV file.

Provide SSH credentials (username and password) or an SSH private key for authentication.

Choose a command template or enter custom commands to execute on each server.

Click "Start Scan" to begin the process. The application will connect to each IP address and execute your commands.

Analyze the results, filter data, and export reports as needed.
